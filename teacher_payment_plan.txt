Student Due System (Working):

Enroll student	BatchController::storeAssignedStudents() calls $this->dueService->generateDueForEnrollment() → Creates StudentMonthlyDue

Teacher Payment System (Current - Manual):
Action	What Happens
Assign teacher	Only adds to batch_teacher pivot table - NO automatic TeachersPayment
Remove teacher	Only removes from pivot table - NO deletion of TeachersPayment
Generate payment	Admin must manually go to /admin/teachers-payments/generate to create TeachersPayment
Key Difference:
- Student dues: Auto-generated on enrollment, auto-deleted on unenroll
- Teacher payments: Manual generation required via separate endpoint
